📊 Mobile Sales Dashboard
📌 Overview

This project is an interactive Mobile Sales Dashboard that provides insights into mobile phone sales performance. It helps analyze sales trends, top-performing brands, customer behavior, and revenue distribution using visualizations.

The dashboard is designed to make data-driven decision-making easier by presenting complex data in a simple and understandable format.

🚀 Features
📈 Sales trend analysis (daily / monthly / yearly)
📱 Top-selling mobile brands and models
💰 Revenue and profit insights
🌍 Region-wise sales distribution
🛒 Customer purchase behavior analysis
📊 Interactive filters for better exploration

📊 Key Insights
Identified top-performing mobile brands based on sales
Observed seasonal trends in mobile purchases
Found high-revenue generating regions
Analyzed customer preferences and buying patterns
